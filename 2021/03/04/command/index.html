VS Code 快捷指令

  1. format 全文: shift + alt + F

  2. format selection: Command + K, Command + F

  3. 快捷添加 Consloe.log:

    ctrl + alt + L 选中变量生成consoles.log

    alt + shift + C 注释所有 consloe.log

    alt + shift + u 启用所有 console.log

    alt + shift + d 删除所有 console.log
